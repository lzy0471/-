MTV 思想
|----manage.py #入口启动
|----app       #应用包
|----__init__.py #初始化模块
|-----configs.py  #配置信息模块
|-----urls.py     #路由和试图映射模块
|-----templates #存放模版目录
|-----admin #存放后台管理系统的视图
|-----api #存放和前端交互API的视图
|-----common  #公共包：人脸识别模块、IP地址解析、表单验证
|----- static #静态资源目录
|-----  UI #自定义UI组件
|-----

启动 python manage.py
